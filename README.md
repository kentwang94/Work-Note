# 技术经验总结文档

这是一个经验知识的技术总结文档项目，支持通过 GitHub Pages 访问。

## 目录结构

- `2018/` - 历史文档归档（2018年及之前的内容）
- `yyyy-mm/` - 月度文件夹，包含每日技术总结（格式：`yyyy-mm-dd.md`）
- `Agents.md` - 仓库信息与维护规则说明

## 新增内容规范

- 每月创建一个文件夹，命名为 `yyyy-mm`（如：`2024-01`）
- 每日新增内容以 `yyyy-mm-dd.md` 格式命名（如：`2024-01-15.md`）

详细维护规则请参阅 [Agents.md](Agents.md)。
